# TODO APP

Esta aplicación permite crear, completar y eliminar tareas, con almacenamiento en el navegador.

## Objetivo de este proyecto

Con este proyecto busqué aprender el uso básico de React, aplicando mis conocimientos en JavaScript y CSS. Este es el primer proyecto que realizo con React.

## ¿Cómo ejecutar el proyecto?

Puedes ejecutar el proyecto desde la rama `gh-pages`, donde encontrarás un deploy en Git. Por otro lado, si deseas ejecutarlo en local, deberás descargar el repositorio desde la rama `main` y ejecutar un `npm run build`.

### Ejecutar en local

1. Clona el repositorio
2. Instala las dependencias:

npm install

3. Ejecuta el proyecto:

npm start

## Tecnologías utilizadas

- React
- JavaScript
- CSS
- Git y GitHub
